

# Aplicacion para almacenar tareas con manejo de Local Storage

## Herramientas

- HTML
- CSS
- Java-Script





```bash
git clone
abrir index.html

```




funcionalidades:

- agregar tareas
- Marcar tareas como completadas
- Eliminar tareas
